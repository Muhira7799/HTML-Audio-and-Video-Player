# HTML Audio and Video Player

## Overview
This project demonstrates how to use HTML5 media elements to embed both video and audio in a web page. It shows the basic implementation of the `<video>` and `<audio>` tags with playback controls.

## Features
- HTML5 video player
- HTML5 audio player
- Playback controls
- Audio loop functionality
- Simple and clean page structure

## Technologies Used
- HTML5

## What I Learned
During this project, I learned how to:
- Embed videos using the `<video>` element.
- Add audio using the `<audio>` element.
- Use the `controls` attribute to enable media playback.
- Apply the `loop` attribute to repeat audio automatically.
- Organize content using semantic HTML elements like `<section>`.

## How to Run
1. Download or clone the project.
2. Open `index.html` in any modern web browser.
3. Play the video or audio using the built-in controls.
